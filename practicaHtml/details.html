Preguntas frecuentes (FAQ)

¿Qué es HTML5?

HTML5 es la última versión del lenguaje de marcado HTML. Introduce nuevas etiquetas semánticas, soporte multimedia y APIs para mejorar la experiencia web.

¿Qué hace la etiqueta <details>?

Permite mostrar u ocultar información adicional a elección del usuario mediante una interfaz nativa del navegador.

¿Se puede mostrar desplegado por defecto?

Sí. Usando el atributo open, el contenido se muestra visible al cargar la página.
